# JavaScript Quiz Game

## Overview
This is a simple quiz game created using JavaScript for *Lab Assignment 4*.  
It runs entirely in the browser console and demonstrates the use of:
- Arrays  
- Loops  
- Conditionals  
- Functions  
- Basic Input/Output (prompt and alert)  
- String methods (toLowerCase and trim)

## Features
- 15 predefined beginner-friendly questions  
- Input cleaned using toLowerCase() and trim()  
- Instant feedback after every answer  
- Correct and incorrect messages  
- Final score displayed at the end  
- Fully runs in the browser console  
- Easy to understand and perfect for beginners  

## How to Run
1. Open Google Chrome (or any browser)
2. Press **F12** and go to the **Console** tab  
3. Copy and paste the entire content of **quiz.js**  
4. Press **Enter** to start the quiz  
5. Answer each question in the pop-up prompts
